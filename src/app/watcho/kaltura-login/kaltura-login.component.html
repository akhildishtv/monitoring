Kaltura Login API

API Name https://restv4-as.ott.kaltura.com/v5_0_3/api_v3/service/ottuser/action/login
Description This API is used in Watcho applications throughout platforms to get a KS token.
When a user logs in to Watcho application, this API is called which generated a token (known as KS string) for that particular user which enables them to watch the video for what they have subscribed for.
