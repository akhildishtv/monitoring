Locations API

API Name http://analytics.mysmartstick.com/api/v1/analytics/channel/newslocationuser
Description This API can be used to fetch all location data along with the number of users who watched the respective news channels.
Start and end parameters specifying the limit in terms of date and time are passed between which the data is fetched.
This API is currenty being used in Elite Panel Analyics Project.
